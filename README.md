### Prerequisites

- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/abuaslamtech/eimaanskincare.git
   cd eimaanskincare
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the landing page.
